What is this?
=============

This presentation on the rings of the planet Saturn is a showcase for
the features of `burst s5 <http://github.com/astraw/burst_s5>`_, a
presentation tool based on HTML and other standard formats using an
open source toolchain.

(Hint: Press F11 on Firefox to get fullscreen mode.)
